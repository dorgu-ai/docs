# Dorgu Platform

The Dorgu Platform provides a web UI dashboard for ClusterPersona visualization and cluster insights.

## Features

- Real-time ClusterPersona resource monitoring
- WebSocket-based live updates
- Cluster state visualization

<Info>
  The platform is in early development. Features and UI are subject to change.
</Info>

## Quick start

```bash
dorgu platform serve
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

See the [platform serve command](/cli/commands/platform) for all available flags.
